# flask-job-board

this is a flask based job board

### usage:

##### 1. Clone this repo

''sh
git clone https://github.com/hadimabroor/flask-job-board
cd flask-job-board
''
##### 2. Install necessary packages

''sh
pip install -r requirements.txt
''

##### 3. Run the code

''sh
gunicorn run:app
''
